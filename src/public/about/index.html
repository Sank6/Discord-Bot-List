About

This is an open-sourced Discord Bot List, written in NodeJS, with an Express JS web server and a discord.js / Klasa bot. The bots are stored in a MongoDB database, but no user data is stored or saved. Cookies are used to store discord's oauth data. If you find any issues or bugs, or have a suggestion, feel free to open an issue.
Some of the icons used in this website are from Icons8.
